# 1.let.const命令

## 作用域

- 全局作用域
- 函数作用域
- 块作用域
  - 一个{ }里为一个块作用域

## let和const用法

- let
  - 和var的作用相似，作用域为块作用域，不能重复定义
- const
  - 用来声明常量，声明时必须赋值，且不能修改，但声明为对象时可修改
