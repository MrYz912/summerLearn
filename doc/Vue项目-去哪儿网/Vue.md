| 名字    | 功能                                    | 简写 |
| ------- | --------------------------------------- | ---- |
| v-on    | 监听事件                                | @    |
| v-bind  | 向子组件传值                            | :    |
| v-for   | 循环                                    |      |
| v-model | 表单输入绑定                            |      |
| v-text  | 将innerText设为对应的值                 |      |
| v-html  | 将innerHTML设为对应的值                 |      |
| v-if    | 返回值决定标签是否展示                  |      |
| v-else  | 和v-if结合使用，紧贴在v-if后面          |      |
| v-show  | 和v-if功能一样，但值为false时，隐藏标签 |      |
